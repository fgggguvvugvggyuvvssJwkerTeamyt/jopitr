# Installation

```sh
# Let's install the bot.
cd $HOME
git clone https://github.com/jwkerteam/jopitr.git
cd jopitr
chmod +x jopitr.sh
chmod 777 auto.sh && sed -i -e 's/\r$//' auto.sh
./jopite.sh install
./jopitr.sh 
# Enter a phone number & confirmation code.
